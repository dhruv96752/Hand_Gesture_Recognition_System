# Hand Gesture Recognition System

A computer vision project built using Python, OpenCV, MediaPipe and TensorFlow.

## Features
- Hand Tracking
- Sign Recognition
- Virtual Mouse Control
- Virtual Painter

## Technologies Used
- Python
- OpenCV
- MediaPipe
- TensorFlow
- NumPy

## Project Structure

- HandTrackingModule.py
- sign recognition.py
- AiVirtualMouseProject.py
- VirtualPainter.py

## How to Run

pip install -r requirements.txt

python "sign recognition.py"
