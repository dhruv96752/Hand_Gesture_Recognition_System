# 🖐️ Hand Gesture Recognition System

A Computer Vision project that uses hand gestures to perform multiple real-time tasks such as:

- Hand Tracking
- Sign Language Recognition
- Virtual Mouse Control
- Virtual Painter

Built using Python, OpenCV, MediaPipe, TensorFlow, and NumPy.

---

## 🚀 Features

### ✋ Hand Tracking
Real-time hand landmark detection and tracking using MediaPipe.

### 🤟 Sign Recognition
Recognizes predefined hand gestures and signs.

### 🖱️ Virtual Mouse
Control the system cursor using hand gestures.

Functions:
- Cursor Movement
- Left Click
- Right Click
- Drag Operations

### 🎨 Virtual Painter
Draw on a virtual canvas using finger movements.

Features:
- Free Hand Drawing
- Color Selection
- Eraser Tool

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Hand-Gesture-Recognition-System.git
```

Move into project directory:

```bash
cd Hand-Gesture-Recognition-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

### Run Sign Recognition

```bash
python sign_recognition.py
```

### Run Virtual Mouse

```bash
python AiVirtualMouseProject.py
```

### Run Virtual Painter

```bash
python VirtualPainter.py
```
